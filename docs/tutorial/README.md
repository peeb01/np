# NP Programming Tutorial Book

Welcome to the NP programming book, designed to guide you step-by-step from absolute basics to advanced features of the NP Language. Each chapter contains navigation links at the bottom to let you move smoothly through the book.

---

## Table of Contents

1. [Chapter 1: Introduction and Environment Setup](chapter_01.md)
   * Introduction to NP and its LLVM compiler architecture.
   * Installing prerequisites and setting up the compiler.
   * Writing your first program (Hello World) and execution modes.

2. [Chapter 2: Variables, Data Types, and Operations](chapter_02.md)
   * Variable declarations and Type Inference.
   * Core types: int, float, string, and bool.
   * Arithmetic/comparison operators and strict type safety.

3. [Chapter 3: Control Flow (Conditionals and Loops)](chapter_03.md)
   * Indentation-based blocks and colons.
   * Decision making with if, elif, and else.
   * Program loops: while loops and for loops (range-based and collection-based).

4. [Chapter 4: Functions and Scoping Rules](chapter_04.md)
   * Declaring functions with fn, arguments, and return types.
   * Calling functions and return value bindings.
   * Variable lifetimes and scopes (Global vs. Local).

5. [Chapter 5: Collections (Array, Dictionary) and Slicing](chapter_05.md)
   * Working with Arrays and built-in methods (.sort, .reverse, .contains).
   * Key-value mappings in Dictionaries.
   * Vectorized Slicing and negative indexing.
   * Element-wise arithmetic and scalar broadcasting.

6. [Chapter 6: Custom Data Models with Structs](chapter_06.md)
   * Declaring structs for complex data modeling.
   * Instance creation and accessing/mutating properties using dot notation (.).
   * Nesting structs inside arrays and dictionaries.

7. [Chapter 7: File Operations and Exception Handling](chapter_07.md)
   * Reading and writing files using read_file and write_file.
   * Safeguarding execution using try, except Exception, and throw.
   * Graceful recovery from runtime errors.

8. [Chapter 8: High-Precision Numerics and Large Projects](chapter_08.md)
   * 128-bit (int128) and 256-bit (int256) signed integers.
   * Multi-file projects using import.
   * Under-the-hood memory management via Automatic Reference Counting (RAII).
